# 청사진 (Blueprint)

## 개요

이 프로젝트는 사용자의 현재 나이, 은퇴 나이 및 인생의 여러 이벤트를 기반으로 시간 경과에 따른 자산 평가액을 예측하는 은퇴 시뮬레이터입니다. 동적이고 이벤트에 기반한 시스템을 사용하여 사용자의 일생에 걸친 투자 전략, 기여금 및 현금 흐름의 변화를 모델링합니다. 모든 데이터는 지속성을 위해 브라우저에 로컬로 저장됩니다.

## 스타일 및 디자인

- **전체 디자인:** TailwindCSS를 활용하여 깨끗하고 데이터 중심적인 현대적 UI를 구현했습니다. 다크 모드를 완벽하게 지원하여 사용자에게 편안한 시각적 경험을 제공합니다.
- **레이아웃:** 주요 기능인 '기본 정보', '시나리오', '시뮬레이션 결과' 섹션으로 명확하게 구분되어 있어 사용자가 쉽게 탐색하고 상호작용할 수 있습니다.
- **컴포넌트:**
    - **시나리오 타임라인:** 사용자가 설정한 금융 이벤트를 **나이와 월 순서**로 정렬하여 수직 타임라인 형태로 시각화합니다. 이를 통해 시간의 흐름에 따른 재정 계획의 변화를 직관적으로 파악할 수 있습니다.
    - **결과 테이블:** 연도별 상세 데이터를 테이블 형식으로 제공하며, 각 행은 클릭 시 해당 연도의 월별 상세 내역을 볼 수 있도록 확장됩니다.
    - **자산 성장 차트:** Chart.js 라이브러리를 사용하여 투자 원금과 수익을 누적 막대그래프로 시각화하여 자산 성장 추이를 한눈에 파악할 수 있게 합니다.
    - **모달 창:** 시나리오 이벤트나 포트폴리오 프리셋을 추가/수정할 때, 화면 중앙에 모달 창을 띄워 사용자 경험의 연속성을 해치지 않고 작업을 수행할 수 있도록 합니다.

## 주요 기능

- **핵심 시뮬레이션 로직 (`simulation.js`의 `simulate` 함수):**
    - **월별 정밀 계산:** 사용자의 현재 나이부터 100세까지, 1년을 12개월로 나누어 모든 계산을 월 단위로 정밀하게 수행합니다.
    - **이벤트 적용:** 매월 시뮬레이션 계산 시작 시점에, 해당 **나이와 월**에 적용되어야 할 모든 활성화된 이벤트를 찾아 즉시 반영합니다.
    - **동적 포트폴리오:** `getActivePortfolio` 함수는 특정 나이와 월을 기준으로, 가장 최근에 적용된 포트폴리오 구성 이벤트를 찾아 현재 투자 전략을 결정합니다.
    - **현금 흐름 처리 순서 (배당금 우선 인출):**
        1.  **투자 및 입금:** 월 정기 투자금과 일시불 입금액을 미투자 현금(uninvested cash)에 더합니다.
        2.  **자산 투자:** 미투자 현금을 현재 포트폴리오 비중에 맞게 각 자산에 투자합니다. 투자 시 자산의 `balance`와 `costBasis`(매입원가)가 함께 증가합니다.
        3.  **수익 발생:** 투자된 자산의 `balance`를 기준으로 월 수익(가격 상승분)이 계산되어 `balance`에만 더해집니다. 
        4.  **배당 발생:** `balance`를 기준으로 세후 배당금이 계산되어, 재투자되지 않고 '월별 인출 가능 현금'(monthlyDividends)으로 별도 관리됩니다.
        5.  **인출:** '월 정기 인출' 필요 시, (4)번의 '월별 인출 가능 현금'에서 우선적으로 사용합니다. 부족할 경우에만 부족분만큼 자산을 매도하여 충당하며, 이 때만 매매차익이 발생하여 양도세 계산 대상이 됩니다.
        6.  **세금 처리:** 연간 누적된 매매차익이 공제액(250만원)을 초과할 경우, 초과분에 대해 양도소득세(22%)가 부과됩니다. 부과된 세금은 포트폴리오 자산의 `balance`와 `costBasis`에서 비례하여 차감됩니다.

- **데이터 지속성 및 공유:**
    - **로컬 저장소:** 모든 입력 정보, 시나리오, 프리셋은 브라우저의 로컬 스토리지에 자동 저장됩니다.
    - **URL 공유:** `pako.js` 라이브러리로 현재 상태를 압축하여 URL 해시(hash)에 담아 다른 사람과 시뮬레이션 결과를 공유할 수 있습니다.

- **상세 결과 및 툴팁:** 결과 테이블의 각 연도 행에 마우스를 올리면, 해당 연도의 포트폴리오 구성과 자산별 수익/배당 내역, 발생 이벤트 목록(적용 월 포함)을 상세한 툴팁으로 보여줍니다.

- **현지화 (Localization):** 모든 UI 텍스트는 `public/strings.js` 파일에서 관리되며, `data-i18n-key` 속성을 통해 동적으로 HTML에 적용됩니다.

- **법적 고지 페이지:**
    - 개인정보처리방침 (`privacy.html`)과 이용약관 (`terms.html`) 페이지를 추가하고 메인 화면에 링크를 연결했습니다.

- **온보딩 가이드:** 신규 사용자를 위해 앱의 핵심 기능을 단계별로 안내하는 온보딩 가이드를 구현했습니다.

- **SEO 및 소셜 공유 최적화:**
    - `index.html`과 `retir-calc.html`에 메타 설명(description)과 오픈그래프(OG) 태그를 추가하여 검색엔진 및 소셜 미디어 공유 시 앱의 정보가 명확하게 표시되도록 했습니다.
    - 소셜 공유 미리보기 이미지(`og-preview.png`)를 생성하여 `public` 폴더에 추가했습니다.

## 현재 계획

**블로그/가이드 섹션 추가**

사용자가 은퇴 계획 및 금융 지식을 심도 있게 이해할 수 있도록 블로그/가이드 섹션을 추가합니다. 이 섹션은 시뮬레이터의 활용도를 높이고 사용자에게 더 큰 가치를 제공하는 것을 목표로 합니다.

**구현 단계:**

1.  **블로그 메인 페이지 생성 (`public/blog.html`):** 모든 가이드 아티클의 목록을 보여주는 랜딩 페이지를 생성했습니다.
2.  **개별 아티클 페이지 생성:** 총 5개의 상세 가이드 아티클 페이지를 `public/blog/` 디렉토리 내에 생성했습니다.
    - `guide-usage.html`: 은퇴 시뮬레이터 사용법 완전 가이드
    - `guide-asset-allocation.html`: 자산 배분 101: 포트폴리오 다각화의 중요성
    - `guide-4-percent-rule.html`: '4% 룰'이란 무엇인가? 은퇴 자금 인출 전략
    - `guide-fire-korea.html`: 한국인을 위한 FIRE 계획 세우기
    - `guide-dividend-reinvestment.html`: ETF 배당금 재투자의 마법: 복리 효과 극대화하기
3.  **메인 네비게이션 링크 추가:** `index.html`의 사이드바에 '가이드' 메뉴를 추가하고 `blog.html`로 연결하여 사용자의 접근성을 확보했습니다.
4.  **`blueprint.md` 업데이트:** 프로젝트 청사진에 신규 기능 추가 이력을 상세히 기록했습니다.
